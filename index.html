I am currently a Junior at Ramapo College of New Jersey.

I will be graduating in the Spring of 2019. Here are some of the courses I have taken so far:

Web Application Development

This course taught me about the in-depth treatment of the critical languages, design patterns, and technologies underpinning all of modern web application development. I was taught the basics of networking, HTML, CSS, and also learned about using JavaScript for client side development (jQuery) and server side (Node.js).

Scientific Programming with Python

This course taught me about common computational approaches to solving scientific problems using Python. I learned about practical scientific applications of Python in data analysis and experimentation and developed programs using stochastic models, experimental probability and statistics, Monte Carlo simulation, regression analysis, and dynamic programming.

Database Design

This course taught me about relational database design and SQL, along with non-traditional databases such as key and document stores. I learned about relational schema design, emphasizing normalization, ER and UML modeling, and aspects of the SQL DDL for creating constraints and relationships between data models. I also learned about SQL and query design.

Computer Graphics

The course taught me about the fundamental techniques and skills of Computer Graphics utilizing WebGL. In addition to pure 3D computer graphics, I learned how to utilize the web - using HTML and JavaScript to deliver their 3D visualizations

Analysis of Algorithms

This course taught me about algorithms. There was an emphasis on the design of efficient algorithms and data structures, proofs of their correctness, and analysis of their complexity. A number of algorithmic concepts and techniques were covered, including recursion, incremental design, divide-and-conquer, greedy algorithms, amortized analysis, and dynamic programming. The algorithms studied include sorting, searching, breadth-first search, depth-first search, minimum spanning trees, shortest paths, network flow, string matching.

Discrete Structures

This course taught me about design ideas that will help me develop and analyze algorithms as well as think about and solve problems in new ways. I learned how to express ideas using mathematical notation and solve problems using the tools of mathematical analysis.
